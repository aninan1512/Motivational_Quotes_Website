# Motivational Quotes Website

A responsive, situation-based motivational quotes website that delivers inspiring quotes tailored to different emotional and life situations. The project focuses on clean UI design, dynamic content delivery, and smooth user interaction using core web technologies.

---

## ğŸ¯ Project Overview

This application allows users to select a life situation such as stress, failure, success, loneliness, or motivation. Based on the selected situation, the website dynamically displays a series of motivational quotes in a shuffled, non-repetitive order, along with visually engaging background images.

The goal of this project is to create a simple yet meaningful user experience while demonstrating strong fundamentals in frontend web development.

## âš™ï¸ How the Application Works

1. The user selects a situation from the dropdown menu  
2. A shuffled list of quotes related to the selected situation is generated  
3. Quotes are displayed one at a time without immediate repetition  
4. Background images change dynamically based on the selected situation  
5. Users can navigate through quotes using the "Next Quote" button
